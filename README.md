# 🌿 NutriTec

Aplicativo de nutrição personalizada com gamificação, metas e planos alimentares para cada perfil.

---

## 🚀 Como publicar (Vercel — recomendado)

1. Acesse [vercel.com](https://vercel.com) e crie uma conta gratuita
2. Clique em **"Add New Project"**
3. Escolha **"Upload"** e arraste esta pasta `nutritec/`
4. Clique em **Deploy**
5. Em ~1 minuto você terá um link do tipo `nutritec.vercel.app`

---

## 🌐 Como publicar (Netlify)

1. Acesse [netlify.com](https://netlify.com) e crie uma conta gratuita
2. Arraste a pasta `nutritec/` diretamente para o painel
3. Aguarde o build automático
4. Seu link estará pronto!

---

## 💻 Como rodar localmente

```bash
npm install
npm run dev
```

Abra [http://localhost:5173](http://localhost:5173) no navegador.

---

## 📱 Funcionalidades

- ✅ Questionário personalizado (9 perguntas)
- ✅ Cálculo de calorias pela fórmula Mifflin-St Jeor
- ✅ 10 planos alimentares com 7 dias completos
- ✅ Dietas especiais: Celíaca, Sem Lactose, Sem Glúten, Diabéticos
- ✅ Criador de dieta personalizada
- ✅ ~90 alimentos em 6 categorias
- ✅ Gamificação: XP, níveis e badges
- ✅ Metas diárias, semanais e mensais
- ✅ Tracker de água e refeições
